# oscillation-des-neutrinos-Stage-code
**L’ensemble des notebooks utilisés pour le stage sur l'oscillation des neutrinos**

## 📋 Contexte
Ensemble de notebooks crée lors de mon stage de Master 1 (ULB, 2025-2026) sous la supervision de Juan Antonio Aguilar Sanchez pour L’étude des oscillations des neutrinos à partir des données
d’IceCube-DeepCore.

## 🎯 Objectif
L’objectif du stage est de travailler sur les données d’IceCube-DeepCore et de suivre l’ensemble de la démarche permettant
de retrouver les paramètres qui définissent les oscillations des neutrinos dans l’approximation du mélange à deux saveurs,
afin d’obtenir finalement les paramètres qui donnent le meilleur ajustement entre la simulation et les données réelles
d’IceCube-DeepCore.

## 📂 Structure
oscillation-des-neutrinos-Stage-code/

├── fluxtask.ipynb # code sur la familiarisation des outils de l'analyse.

├── hypersurface.ipynb # code de l'analyse final pour récupérer les paramètres ajustés avec la correction des hypersurface.

├── sanshypersurface.ipynb # code de l'analyse final pour récupérer les paramètres ajustés sans la correction des hypersurface.

└── README.md
